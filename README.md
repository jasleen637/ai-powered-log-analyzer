# AI-Powered Log Monitoring System

An AI-powered backend system that analyzes application logs using Generative AI to detect errors, classify severity, and provide actionable debugging insights.

## Overview

Modern applications generate large volumes of logs that are difficult to analyze manually. This project leverages Generative AI to automatically:
- Summarize logs
- Identify errors
- Classify severity (LOW / MEDIUM / HIGH)
- Suggest possible fixes

Goal: Reduce debugging time and improve developer productivity

## Features

🔍 Log Analysis – Processes raw logs and extracts meaningful insights
⚠️ Error Detection – Identifies key errors and anomalies
📊 Severity Classification – Categorizes issues by impact
💡 Actionable Suggestions – Recommends possible fixes
⚡ REST API Support – Simple API to analyze logs
